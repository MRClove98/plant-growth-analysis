# plant-growth-analysis
This is a plant growth statistics collector, where you can input the type of plant, number of plant, and the height of the plant. The program will tell you the tallest plant, shortest plant and the average height of the plants. We made this program easy and fun to use!

## Features
-Input plant's type
-Input amount of plants
-Input height of the plant
-Gives back accurate report for the tallest plant, shortest plant and the average height of the plants
